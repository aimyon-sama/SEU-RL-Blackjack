# BlackJack 环境

## 简述

这是一个简化了的多玩家21点游戏，具体玩家数量不定，每一整局游戏流程如下

- 将所有玩家随机排序，作为这一局的座次。
- 循环进行如下的每一小局
    - 玩家下注
    - 庄家给每个玩家发两张明牌，再给自己发一张明牌，一张暗牌
    - 庄家检查自身是否持有BlackJack，如有，除了持有BlackJack的玩家可以拿回赌注，其他所有玩家立即输掉这一小局，这一小局也结束。否则所有玩家按顺序依次行动。
        - 当轮到某玩家时，该玩家现决定是否 “Double” 或者 “Surrender”，如选择两个行动中的某一个，则经过固定行动后行动阶段结束。否则进入下一阶段。
        - 玩家从 “Hit” 或者 “Stand” 中选择一个动作，选择动作 “Stand” 后行动阶段立刻结束。
    - 等所有玩家完成行动后，庄家展示明牌，并按照点数小于17发牌，大于17停牌的策略给自己发牌，如爆牌则在场玩家全部胜利，否则进入下一步。
    - 庄家与在场的玩家进行比大小，玩家大则赢取赌注等量的筹码并拿回赌注，相同则拿回赌注，否则失去赌注。
- 最终按照玩家出局顺序和持有筹码总数排名。

上述流程中，一旦玩家爆牌，玩家立刻离开这一小局，并且失去所有赌注。

- 爆牌：指手牌点数大于21。
- 点数：将扑克牌面数值2-9视作2-9，10 J Q K 视作10，A 视作 1或者11，按总计点数不爆牌的最大值计算。（这意味着至多只有一张 A 可以视作11）
- 在场：玩家不爆牌，或者不Surrender，且本小局开始前剩余筹码为正。
- BlackJack：指初始两张牌是 A 和 10 J Q K中的一张组成的牌型，这被规定为最大的牌型，大于其他点数为21点的牌型。当持有BlackJack赢得一局时，玩家会赢得1.5倍的赌注。

## 行动定义
- Double：玩家将赌注提高到两倍（筹码允许的情况下），并且必须要且只能要一张牌，然后小局结束。当筹码量不足时，该动作不会生效，自动进入下一决策阶段。
- Surrender: 玩家放弃这一局，立刻退出这一小局，并拿回一半的赌注。
- Continue: 非Double和Surrender情况下的占位符。
- Hit：要一明张牌到手牌中。
- Stand: 停止要牌

## 决策阶段
每一小局有三个决策阶段
- 下注：返回一个int型的数字，至少下注1，至多下注持有筹码的总量，其他输入数字会被限制到这一范围。
- 首次行动：返回一个str，可选值 "Double" "Surrender" "Continue"
- 后续行动：返回一个str，可选值 "Hit" "Stand"

## 信息
不同阶段环境会告诉玩家不同的信息，包括：座次、初始持有筹码量、场面所有的明牌、前序玩家在本小局内的历史行动等。
信息的具体格式请参见源码以及demo。
